Версія з інтернету
 термінал та перейдіть до локальної копії вашого проекту.
 команду git add [ім'я файлу], щоб додати зміни до стадії Git.
 команду git commit -m "повідомлення коміту", щоб зберегти зміни у Git.
 команду git push [назва репозиторію] [назва гілки], щоб пушити зміни у віддалений репозиторій. Назва репозиторію - це назва віддаленого репозиторію, а назва гілки - назва гілки, до якої ви пушите зміни.
