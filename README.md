# GIT_REPO_HOME_WORK1
Відкрийте термінал та перейдіть до локальної копії вашого проекту.
Використайте команду git add [ім'я файлу], щоб додати зміни до стадії Git.
Використайте команду git commit -m "повідомлення коміту", щоб зберегти зміни у Git.
Використайте команду git push [назва репозиторію] [назва гілки], щоб пушити зміни у віддалений репозиторій. Назва репозиторію - це назва віддаленого репозиторію, а назва гілки - назва гілки, до якої ви пушите зміни.
